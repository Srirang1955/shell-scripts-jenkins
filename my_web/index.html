ABOUT

From Automotive Software Validation to Cloud Engineering

My career began in automotive software validation, where I worked on premium infotainment systems, ensuring quality across Bluetooth, Wi-Fi, Apple CarPlay, Android Auto, ECU communication, firmware flashing, and end-to-end validation. Working closely with developers and system engineers gave me a strong appreciation for software reliability, testing discipline, and engineering best practices.

Over time, I became increasingly interested in how applications move from source code to production. That curiosity led me into the world of DevOps, where I started learning Linux, Docker, Kubernetes, AWS, Jenkins, Git, Infrastructure as Code, and cloud-native application deployment.

Today, my goal is to combine my testing background with DevOps practices to build scalable, automated, and reliable systems.
